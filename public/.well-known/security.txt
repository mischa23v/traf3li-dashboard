# RESPONSIBLE DISCLOSURE GUIDELINES
# ============================================================================
#
# We take security seriously. If you discover a vulnerability:
#
# 1. Report it to security@traf3li.com with:
#    - Detailed description of the vulnerability
#    - Steps to reproduce
#    - Potential impact assessment
#    - Any proof-of-concept code (if applicable)
#
# 2. Our commitment:
#    - Acknowledge receipt within 24 business hours
#    - Provide initial assessment within 72 hours
#    - Keep you informed of remediation progress
#    - Credit you in our acknowledgments (if desired)
#
# 3. Please:
#    - Allow reasonable time for remediation before disclosure
#    - Do not access or modify user data
#    - Do not perform denial-of-service attacks
#    - Do not social engineer our employees
#
# We will not pursue legal action against researchers who:
#    - Follow responsible disclosure practices
#    - Act in good faith
#    - Do not cause harm to our users or systems
#
# Thank you for helping keep Traf3li and our users secure.
